Nesse exemplo estamos usando o mongodb atlas para criar o banco de dados na nuvem

Para inicializar primeiro criamos o arquivo .gitignore contendo node_modules/

depois inicializamos o comando npm init (setar no entry point o nome do arquivo js de inicio, nesse caso server.js)

depois instalamos as seguintes dependencias

npm i 
express -> principal framework web para backend
express-validator -> para validação de dados, por exemplo se fizermos uma requisicao POST para uma API, se tiver campos faltantes ele vai gerar um erro
bcryptjs -> usado para encriptacao de passwords
config -> para variaveis globais
gravatar -> para perfis do gravatar, pode usar um email associado com gravatar para cadastrar
jsonwebtoken -> estamos JWT para passar token para validacao
mongoose -> uma camada que fica acima do banco de dados para podermos interagir com o banco
request -> usado para fazer requests HTTP para outra API, vamos usar para fazer requests para a API do GitHub
